# 🔐 MITM Attack Simulation

## 📌 Overview
This project demonstrates a **Man-in-the-Middle (MITM) attack** in a controlled environment to understand how network communication can be intercepted and analyzed.

## 🎯 Objective
- To study the working of MITM attacks  
- To analyze how data can be intercepted between two communicating parties  
- To create awareness about network security vulnerabilities  

## ⚙️ Methodology
- Set up a controlled network environment  
- Performed interception using MITM techniques  
- Monitored and analyzed the transmitted data  

## 🛠️ Tools & Technologies
- Networking tools (e.g., packet analyzers)
- System environment for simulation
